# Tartsi Documentation

Tartsi is a teaching-focused React application that generates ritual-style magic circles from a geometric symbolic language.

## What this documentation is for

- It teaches complete beginners how the project is structured.
- It explains the wave-function-collapse-inspired generator without assuming prior graphics knowledge.
- It documents the quality and security systems that keep the repository healthy.

## Start here

1. Read [Getting started](./guide/getting-started.md).
2. Continue to [How the generator works](./guide/how-the-generator-works.md).
3. Try the hands-on tutorial: [Build your first magic circle](./tutorials/first-magic-circle.md).
