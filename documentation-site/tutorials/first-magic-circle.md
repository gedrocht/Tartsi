# Build Your First Magic Circle

This tutorial walks through the first meaningful thing a beginner should try.

## Step 1: Start the app

```bash
npm run dev
```

Open the local development address that Vite prints in the terminal.

## Step 2: Change the seed phrase

Replace the default seed phrase with something memorable like:

```text
Celestial-Library-Tutorial
```

Because the generator is deterministic, the same seed always reproduces the same circle.

## Step 3: Increase complexity

Raise the complexity slider and watch the outer rings gain more segments and denser symbolic structures.

## Step 4: Export the result

Use the export button to download the generated SVG file.

## What you learned

- Seed phrases control deterministic randomness.
- Ring count changes the overall structure.
- Complexity affects segment density.
- The renderer can export crisp vector artwork.
