# External Libraries

This page explains why each major external library exists in the project and links to its official documentation.

## Runtime libraries

- [React](https://react.dev/): builds the component-based user interface.
- [React DOM](https://react.dev/reference/react-dom): mounts React into the browser.

## Build and language tooling

- [TypeScript](https://www.typescriptlang.org/docs/): provides static typing and safer refactoring.
- [Vite](https://vite.dev/guide/): powers local development and production builds.

## Testing

- [Vitest](https://vitest.dev/guide/): runs unit and component tests.
- [Testing Library](https://testing-library.com/docs/react-testing-library/intro/): tests UI from the user’s perspective.
- [jest-axe](https://github.com/NickColley/jest-axe): runs automated accessibility checks.
- [Playwright](https://playwright.dev/docs/intro): runs end-to-end browser tests.

## Code quality and documentation

- [ESLint](https://eslint.org/docs/latest/): enforces coding standards.
- [Prettier](https://prettier.io/docs/en/): checks formatting consistency.
- [TypeDoc](https://typedoc.org/): generates API documentation from source comments.
- [VitePress](https://vitepress.dev/guide/what-is-vitepress): powers the serveable documentation site.
- [DokuWiki](https://www.dokuwiki.org/dokuwiki): powers the separate beginner-friendly wiki layer.
